# Campsoon 用户使用指南

Campsoon 是一个帮助用户自动监控BC Parks 营地空位，并添加到购物车或者自动支付的Chrome 插件。

先下载最新版本的 Campsoon 插件：[https://dub.sh/x2yQGXT](https://dub.sh/x2yQGXT)

> 图片占位：在这里放一张 Campsoon 扩展主界面的截图。

## 手动安装 Chrome 插件

如果你收到的是 Campsoon 插件压缩包，可以这样安装：

1. 打开下载链接 [https://dub.sh/x2yQGXT](https://dub.sh/x2yQGXT)，下载最新的 Campsoon 插件压缩包。
2. 解压压缩包，放到一个固定位置，例如桌面或 Documents 文件夹。
3. 打开 Chrome，在地址栏输入 `chrome://extensions/`。
4. 打开右上角的「开发者模式」。
5. 点击「加载已解压的扩展程序」。
6. 选择刚刚解压出来的 Campsoon 插件文件夹。
7. 安装完成后，点击 Chrome 右上角的扩展图标，打开 Campsoon。

使用 Campsoon 扫描时，必须保持 Chrome 浏览器打开，并确保 Campsoon 插件正在运行。请取消电脑自动休眠，或在扫描期间不要让电脑进入休眠状态；

## 重要概念

### 账号

使用 Campsoon 前需要先登录账号。账号用于保存 Trip、同步点数余额，并接收预订结果通知。

同时，你也需要在 Chrome 里登录 BC Parks 账号，并提前填好 BC Parks 里的用户资料。只有 BC Parks 账号已登录、信息已准备好，插件才能帮你自动 Reserve 或 Auto-pay。

### Trip

Trip 是一次营地搜索任务，包含你想去的营地、日期、筛选条件，以及发现空位后的处理模式。

如果一个 Trip 里有多个营地，系统会按你设置的顺序扫描，排在前面的营地优先级更高。

### Match

Match 是指系统发现了符合你 Trip 条件的空位。发现 Match 后，Campsoon 会根据你选择的模式继续处理。

## 三种处理模式

### Alert Only

Alert Only 是免费功能。系统发现空位后，会发送邮件提醒，但不会帮你预订，也不会付款。

适合只想收到空位通知，然后自己手动进入 BC Parks 操作的用户。

### Auto-reserve

Auto-reserve 会在发现空位后，自动尝试 Reserve，并把营地加入购物车。

成功 Reserve 后，你会收到邮件提醒。你只需要回到已经打开的 BC Parks 标签页，确认信息并点击支付。

这种模式适合你想先二次确认营地是否符合偏好，再决定是否付款的情况。使用时建议你在电脑旁边，方便及时完成付款。

### Auto-pay

Auto-pay 会在发现空位后，自动完成预订和付款，全程不需要你手动操作。预订成功后，你也会收到邮件提醒。

使用 Auto-pay 前，需要在 Campsoon 的 Park Payment 页面填写信用卡号、持卡人姓名、有效期、CVV 和账单地址。

信用卡信息会完全加密存储在你的本地电脑中，不会上传到 Campsoon 服务器。Campsoon 服务器不会记录任何信用卡相关信息。盗用信用卡是违法犯罪行为。如果你仍然不放心，建议使用预付费信用卡，把风险降到最低。

## 使用流程

1. 打开 Campsoon 插件并登录账号。
2. 创建一个 Trip。
3. 添加想去的营地，并按偏好排序。
4. 添加可以接受的入住日期。
5. 选择处理模式：Alert Only、Auto-reserve 或 Auto-pay。
6. 检查筛选条件，例如是否排除 walk-in 营地或 double site。
7. 启动扫描，并保持 Chrome 和电脑正常运行。
8. 等待邮件提醒或预订结果。

## 收费和点数

Alert Only 免费，只发送空位邮件提醒。

Auto-reserve 和 Auto-pay 是收费功能，只有成功完成预订后才会扣除点数。每次成功预订扣除 1000 points。

当前点数套餐：

- 1000 points：CAD 10
- 3000 points：CAD 20

收费主要是为了避免无限制扫描影响大家抢营地的成功率，同时覆盖开发、服务器和维护成本，让服务可以长期稳定运行。

## 使用建议

- 优先选择你真正愿意去的营地，不要一次添加太多不相关目标。
- 日期越灵活，成功机会越高。
- 使用 Auto-pay 前，请确认 BC Parks 账号、付款信息和出行计划都准备好。
- 如果一个 Trip 长时间没有结果，可以尝试放宽日期或增加备选营地。
- 预订成功后，请及时查看 BC Parks 的确认信息。

## 常见问题

### 发现空位后一定能订到吗？

不能保证。热门空位可能在很短时间内被其他人抢走，最终结果以 BC Parks 确认为准。

### 扫描会一直运行吗？

只有在电脑开机、Chrome 打开、Campsoon 插件正常运行时，扫描才会继续。电脑休眠后，插件不会继续工作。

### 我的信用卡信息会上传到服务器吗？

不会。Park Payment 里的信用卡信息会完全加密存储在本地电脑中，不会上传到 Campsoon 服务器，服务器也不会记录任何信用卡相关信息。

## 重要提醒

Campsoon 是辅助工具，不能保证一定订到营地。请在出行前仔细核对日期、营地、人数、车辆和付款信息。
